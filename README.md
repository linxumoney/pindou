# 拼豆图片转换器

一个纯前端、可静态部署的拼豆图片转换网站。用户上传图片后，网站会在浏览器本地完成图片压缩、像素化、颜色量化、拼豆色板匹配、图纸预览、用色统计和高清 PNG 导出。

## 功能

- 支持 JPG、PNG、WEBP、GIF、BMP 等常见图片上传
- 支持点击上传和拖拽上传
- 上传后显示原图预览
- 大图自动降采样到合理尺寸，避免处理卡顿
- 支持按格数或按成品尺寸生成图纸
- 支持 2mm / 5mm 拼豆规格
- 支持锁定原图宽高比例
- 支持裁切填满或完整留白
- 支持透明、白色、自定义背景
- 支持原图近似色模式和拼豆色板匹配模式
- 内置常用拼豆色板，颜色包含编号、名称和色值
- 支持限制最大颜色数量
- 支持显示/隐藏网格线
- 支持显示/隐藏颜色编号
- 支持缩放预览和滚动查看大图纸
- 自动统计每种颜色的预计颗数
- 支持导出高清 PNG 图纸，适合保存或打印

## 本地运行

这是一个无后端、无构建依赖的静态项目。

直接用浏览器打开：

```text
bead-studio/index.html
```

也可以用任意静态服务器运行，例如 VS Code Live Server、Nginx、GitHub Pages、Vercel、Netlify。

## 部署

将 `bead-studio` 目录作为静态网站目录部署即可。

主要文件：

```text
bead-studio/
  index.html   页面结构
  styles.css   页面样式和响应式布局
  app.js       图片处理、拼豆算法、Canvas 渲染、PNG 导出
  README.md    项目说明
```

## 使用方式

1. 点击或拖拽上传图片。
2. 选择按格数或按成品尺寸生成图纸。
3. 根据实际拼豆选择 2mm 或 5mm。
4. 调整色彩数量、图片适配方式、背景和色彩模式。
5. 查看图纸预览和用色清单。
6. 需要打印时开启导出编号，然后点击“下载 PNG”。

## 浏览器兼容性

支持现代浏览器：

- Chrome
- Edge
- Safari
- Firefox

图片转换全部在浏览器本地完成，不会上传到服务器。

## 已知限制

- 超过 `200 × 200` 的拼豆图纸会被限制到最大 200 格，避免浏览器长时间卡顿。
- 内置色板是常用拼豆色的近似集合，不对应某一个品牌的完整官方色号。
- GIF 会按浏览器解码结果处理首帧或默认帧，不生成动画图纸。
